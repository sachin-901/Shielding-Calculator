Calculation Formula
B = [ P × (dtotal)2 ] / [ W × 1000 × U × T ]
Where:
B = Required transmission factor
P = Permissible dose limit (mR/week)
W = Workload (cGy/week)
1000 = Conversion factor (1 cGy ≈ 1000 mR)
U = Use factor
T = Occupancy factor
dtotal = Total distance to occupied area (meters)

n = -log10(B)
Required Thickness = n × TVL

Note on Outer Shielding:
If extra shielding is added to the outer side, dtotal becomes dinner + twall + textra. Because distance increases, the required attenuation slightly decreases. The calculator solves this iteratively to find the exact required extra thickness.
