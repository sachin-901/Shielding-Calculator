Calculation Formula
B = [ P × (dtotal)2 ] / [ W × 1000 × U × T ]

Where:

  • B = Required transmission factor
  • P = Permissible dose limit (mR/week)
  • W = Workload (cGy/week)
  • 1000 = Conversion factor (1 cGy ≈ 1000 mR)
  • U = Use factor
  • T = Occupancy factor
  • dtotal = Total distance to occupied area (meters)

Required Attenuation:
n = -log10(B)
Required Thickness = n × TVL

Note:
  • Outer Wall Placement: If extra shielding is added to the outer side of the bunker, the total distance (dtotal) to the occupied area increases by the thickness of the newly added material. Because radiation follows the inverse square law, this increased distance slightly reduces the required attenuation. The calculator uses an iterative mathematical loop to perfectly balance this increased distance against the extra TVLs required.
  • Density Scaling: The Custom TVL values in the reference table are generated using linear density scaling based on ordinary concrete (2.35 g/cm³). The formula is: Custom_TVL = Concrete_TVL × (2.35 / Custom_Density). Note that actual TVL values can vary slightly due to material composition (e.g., high-Z vs low-Z materials causing spectral changes), so standard literature values should be confirmed for final clinical commissioning.
